# AI Implementation Plans

This folder contains step-by-step implementation plans for specific tasks or features.

## Instructions
- Before starting a complex task, create a new plan: `YYYYMMDD_task_name_plan.md`.
- Update the plan as you progress, or create a "V2" if the approach changes significantly.
- Reference these plans in the `STATE.md` file located in the `AI/` folder at the workspace root.
